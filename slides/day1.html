Angular core concepts

MVC framework pattern

Angular approach

  • M - Services(remote, local), controllers(local)
  • V - HTML, directives (ng-attributes)
  • C - Controllers

View

  • Data binding: {{ someData }}
  • Directives: ng-app, ng-controller, ng-repeat, ng-click, ng-show/ng-hide, ng-class, ng-change...
                        
                        
                            

                                {{ ctrl.message }}

Controller

  • $scope: glues angular app with the view. $scope.message will be in the view {{ message }}
  • Store and manipulate local data.
                        
function myController() {
    vm = this
    vm.message = 'Hello world'
}

But something is missing?!

We need to define a module

                        
angular.module('myApp', []).controller('myController', [myController]);

Services

  • The model
  • $http: get data online
                        
angular.module('myApp', []).service('myService', function($http) {
    $http.getAll = function() {
        var api = 'someUrl.com'
        return $http.get(api + '/alldata')
    }
})
